## 📌 Project Overview
This VAT Compliance Tracker was developed using a dataset sourced from Kaggle. The data was cleaned and transformed using Power Query, including the conversion and formatting of monetary values into Nigerian Naira (₦).
The dashboard is designed to monitor VAT compliance across customers, track VAT attributable to individual customers, and provide actionable insights through interactive Excel visualizations and reporting. The analysis includes total sales and their corresponding VAT obligations, customer-level VAT performance, compliance status tracking, and the percentage of filing periods completed before their due dates. These insights help users evaluate VAT compliance trends, identify potential risks, and support informed decision-making.
## 🔄 Data Cleaning & Transformation
The dataset was cleaned and transformed using Power Query to ensure it was suitable for analysis. Duplicate records and inconsistencies were removed, and missing values were handled appropriately. Date and numerical fields were also standardized to ensure accuracy and consistency across the dataset.
